# Microsoft Teams Countdown

Countdown for Microsoft Teams

## Install

```bash
$ composer install
```

## Usage

```bash
$ ./ms-teams-countdown -d 2018-04-30 -wu https://outlook.office.com/webhook/XXXXXXXX -e "the END"
```

Result : 

```
48 days until the END
```
